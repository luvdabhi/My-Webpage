Some HTML tags for making a website

  • ul,ol,dl: are used to create lists
  • b: this tag is used to bold text
  • Div: used to divide HTML document
  • Body: this is the main body of the HTML document
  • h1,h2,h3...: This is the header text that is bolder larger than normal text
  • p: used to write paragraph in the document
  • img: used to insert an image in the document
  • br: used to make line spaces
  • table: creates tables
  • tr/td: creates rows/columns for a table
  • a: This tag is used to create hyperlinks
